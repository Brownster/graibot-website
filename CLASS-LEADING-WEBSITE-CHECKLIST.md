# GraiBot Website Checklist

## Completed in this pass

- Replace stub blog post pages with full article content.
- Add a dedicated `blog/index.html` hub page.
- Align homepage blog dates to real March 2026 publish dates instead of future dates.
- Improve page-level SEO metadata across homepage, blog posts, and service pages.
- Add richer structured data: `Organization`, `WebSite`, `Blog`, `BlogPosting`, `Service`, and `FAQPage`.
- Add breadcrumb navigation, stronger internal linking, and a fuller sitemap with `lastmod`.

## Highest-priority next steps

- Set up Google Search Console and Bing Webmaster Tools, then submit `https://graibot.com/sitemap.xml`.
- Add first-party analytics and conversion tracking for demo requests, email clicks, and product logins.
- Create dedicated landing pages for the highest-intent terms you want to win, not just broad category pages.
- Publish at least 10 to 20 genuinely useful blog posts and case-study pages around one clear topical cluster.
- Add customer proof: logos, case studies, security posture, and measurable outcomes.
- Create a proper social sharing image in PNG format for homepage, services, and blog posts.

## Technical upgrades for a class-leading site

- Add Cloudflare `_headers` rules for long-lived asset caching and stronger security headers.
- Verify clean URL behavior and, if needed, add `_redirects` to enforce one canonical URL per page.
- Compress and resize all imagery, then use raster OG images where platforms expect them.
- Run Lighthouse, WebPageTest, and Search Console Core Web Vitals reviews on live pages.
- Add an automated link checker and HTML validation step to deployment.
- Add `lastmod` automation so the sitemap is updated by content changes, not by hand.

## Content and SEO upgrades

- Build a keyword map that assigns one primary search intent per page and avoids cannibalization.
- Expand service pages with stronger proof, implementation detail, and use-case specificity.
- Add author pages or editorial trust signals if content marketing becomes a major acquisition channel.
- Add comparison pages, buyer guides, and implementation pages for mid-funnel search intent.
- Add FAQ sections only where they answer real buyer questions and can earn rich results.
- Strengthen internal linking between homepage, service pages, blog posts, and future case studies.

## Conversion and credibility upgrades

- Add clear demo-booking CTAs with a short form instead of mailto-only conversion paths.
- Add a product walkthrough, screenshots, or short video clips showing the platform in use.
- Add a security and compliance page covering data handling, tenancy, and testing methodology.
- Add customer success stories with concrete numbers: defect reduction, faster release confidence, reduced leakage risk.
- Add trust-building microcopy around who the product is for, what integrations exist, and how onboarding works.

## Operating model

- Publish on a fixed cadence so the site keeps earning freshness and topical authority.
- Review search queries monthly and update pages based on impressions, CTR, and ranking gaps.
- Feed production incidents and sales objections back into content ideas and landing-page revisions.
